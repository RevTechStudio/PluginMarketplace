# Design

`detailed-design`スキルを使用して要件定義書から詳細設計書を作成する。  
ドキュメント作成は`document-conventions`および`development-conventions`に準拠すること。  

- **要件定義書**: `./.spec/requirements.md`

詳細設計書は以下のパスに保存：

- `./.spec/design.md`

作業完了後、ユーザーに提示する:

```markdown
詳細設計書を作成しました。

次は`/tasks`コマンドを実行して実装計画書を作成してください。
```

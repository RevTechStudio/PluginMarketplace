# Analyze

`project-analyzer`スキルを使用してプロジェクトを分析する。  
ドキュメント作成は`document-conventions`および`development-conventions`に準拠すること。  

分析結果は以下のパスに保存：

- `./.spec/project-analysis.md`

作業完了後、ユーザーに提示する:

```markdown
解析結果を出力しました。

次は`/brainstorm`コマンドを実行してアイディアの深堀を行ってください。
```
